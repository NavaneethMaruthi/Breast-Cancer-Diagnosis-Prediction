Breast Cancer Diagnosis Prediction
1. Overview

This project uses the Breast Cancer Wisconsin (Diagnostic) dataset to predict whether a tumor is malignant or benign.
We apply three machine learning models and compare their performance.

2. Dataset

569 samples
32 features extracted from digitized FNA images
Target label: M (malignant) or B (benign)

3. Goal

Build models to predict tumor diagnosis
Evaluate accuracy and misclassification
Check for overfitting/underfitting
Analyze misclassified samples
Visualize data and results

4. Models Used

- Logistic Regression

- k-Nearest Neighbors (k-NN)

- Decision Tree
